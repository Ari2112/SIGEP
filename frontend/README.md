# SIGEP Frontend

Frontend del Sistema Integral de Gestión de Personal desarrollado con React + Vite.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`
